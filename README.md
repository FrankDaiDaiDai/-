# 量化因子研究与回测框架

这个项目构建了一个模块化的量化研究闭环，涵盖了从底层数据清洗、多因子建模、独立因子评估到组合优化回测的全流程。目前代码库的核心研究对象是基于 **Fama-French 三因子模型**剥离出的**特质波动率（Idiosyncratic Volatility）**因子，并验证其在 A 股市场的 Alpha 效应。

### 研究逻辑与技术实现

项目的架构设计遵循“解耦”原则，将数据逻辑与策略逻辑完全分离，方便进行横向扩展：

1. **动态数据引擎 (`data_enigine.py`)**
   系统接入了 `rqdatac` 接口，并针对 A 股市场特有的生存者偏差问题，实现了中证 1000 等指数成分股的动态追踪。为了保证回测的精确度，引擎摒弃了简单的后复权数据，而是通过 `ex_factor` 手动计算前复权价格，确保了回测时技术指标的真实性与严谨性。

2. **多因子剥离与特征提取 (`factors/`)**
   特质波动率因子是通过 Fama-French 3-Factor 模型回归后的残差项推导而来的。回归方程如下：
   $$R_{i,t} - R_{f,t} = \alpha_i + \beta_{MKT}MKT_t + \beta_{SMB}SMB_t + \beta_{HML}HML_t + \epsilon_{i,t}$$
   其中因子值定义为残差 $\epsilon_{i,t}$ 的滚动标准差。项目在 `cal_factors.py` 中内置了工业级的预处理流程，包括标准差倍数缩尾（Winsorization）和横截面 Z-Score 标准化，消除了极值和量纲对模型的影响。

3. **独立因子评估 (`evaluate_factors.py`)**
   区别于市面上很多将评估逻辑嵌入回测过程的做法，本项目提供了一个独立的评估工具。在进入交易模拟之前，研究员可以先通过该文件对因子进行纯粹的统计校验。它会自动计算因子的每日 Rank IC（秩相关系数）以及长期 IR（信息比率），帮助判断信号的预测强度和稳定性，从而有效过滤掉虚假的“噪声因子”。

4. **组合优化与回测执行 (`strategy_module.py` & `backtest.py`)**
   在信号生成阶段，策略不仅支持简单的等权重打分，还引入了基于 **SLSQP 算法**的组合优化器。在选出 Top-N 股票后，系统会尝试求解最大化夏普比率的最优权重分布。回测引擎则模拟了真实的交易约束，考虑了 **0.1%** 的手续费率、涨跌停买入限制以及持仓资产的价格漂移。

### 项目结构说明

* **data/**: 存放本地缓存的行情数据与成分股掩码。
* **factors/spec_vol_factor.py**: Fama-French 模型的回归逻辑与特质波动率计算核心。
* **evaluate_factors.py**: 独立的因子诊断工具，用于产出 IC/IR 等统计指标。
* **strategy_module.py**: 策略逻辑层，包含因子排序信号与 SLSQP 组合优化逻辑。
* **backtest.py**: 主运行程序，执行回测循环并在 `output/` 目录下生成对数坐标的净值曲线。

### 快速上手

1. **环境准备**：安装 `pandas`, `numpy`, `rqdatac`, `scipy`, `sklearn` 等必要库。
2. **配置鉴权**：在根目录创建 `.env` 文件，填入你的 Ricequant 凭证：`PASSWD=your_password`。
3. **参数设置**：根据需要在 `settings.yaml` 中修改回测的起止日期与目标股票池。
4. **运行流程**：
   - 首先运行 `python evaluate_factors.py` 检查因子有效性。
   - 然后执行 `python backtest.py` 开启策略回测并查看绩效报告。
